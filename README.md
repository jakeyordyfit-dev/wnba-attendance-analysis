# WNBA Attendance Analysis Pipeline

## Overview
Sports business analytics pipeline analyzing 27 years of WNBA game attendance data across 6,160 games (1997-2024). Identifies long-term attendance trends, peak attendance games, and top franchises by average home attendance.

## Tools Used
- Python (pandas)
- Tableau Public

## Analysis
- WNBA attendance trend from 1997-2024 including COVID impact and 2024 surge
- Top 10 highest attendance games in WNBA history
- All-time franchise rankings by average home attendance

## Key Insight
The 2024 season saw a dramatic attendance spike — a 60% increase from the prior year — driven by generational talent entering the league, making it the most attended season since the late 1990s.
